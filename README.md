# 💳 Credit Card Fraud Detection System

A Machine Learning based web application that detects fraudulent credit card transactions using trained classification models. The project uses data preprocessing, feature scaling, and machine learning techniques to predict whether a transaction is legitimate or fraudulent.

## 🚀 Project Overview

Credit card fraud is a major financial problem. This project builds an ML-based fraud detection system that analyzes transaction features and predicts fraudulent activities.

The trained model is integrated with a Streamlit web application where users can enter transaction details and get instant predictions.

## ✨ Features

- 🔍 Fraudulent transaction prediction
- 📊 Model performance evaluation
